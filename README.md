# UiPathRED
NodeRED packaged with UiPath Orchestrator nodes

## To Install:
Run Install.ps1 in Powershell.

## To Run Node-RED:
Double-click or run Run.lnk.

### Notes:
The following will be installed in addition to Node-RED:
* Chocolatey
* NodeJS
* Git
* node-gyp
* node-red-admin
* node-red-dashboard
