# UiPathRED
NodeRED packaged with UiPath Orchestrator nodes

## To Install:
Run Install.cmd as Administrator

## To Run Node-RED:
Double-click or run Run.cmd

### Notes:
The following will be installed in addition to Node-RED:
* Chocolatey
* NodeJS
* Git
* node-gyp
* node-red-admin
* node-red-dashboard
